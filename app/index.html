Public Information Security Systems

Public Information Security Systems monitor and display any and all information transfers conducted on local wireless protocols. This allows all citizens to help monitor their peers for dangerous knowledge transfer. In the event that an attempt is made to access a deviant information source and alert will be triggered.

The Public Information Security System can also be used to display promotional messages from the Citizens Office of Communications Safety.

example installation one

The Citizens Office of Communications Safety is mandated to install a Pubic Information Security System in all public spaces within the vicinity of any wireless information communications protocol.
